Description du projet :

Projet de fin d'année réalisé par Cyriac Lenoir, Margaux Caradant, Harry Mbengmo. L'objectif est de concevoir une application WEB autour des données sur les infrastructures de recharge pour véhicules électriques (IRVE).
L'application propose un visuel (tableau/carte), des statistiques dynamiques par département, ainsi que les résultats du projet IA. Ce projet inclut également un système de pagination, d'auto-complétion et une partie back.
